# 現実派 不動産収支シミュレーター(デプロイ版・フリーミアム対応)

Vercelにそのままデプロイできる構成です。Free/Proの2プランで、
ライセンスキー認証によりPro機能が開放されます。

## プラン構成

| 機能 | Free | Pro |
|---|---|---|
| かんたんモード・プリセット・信号機診断 | ✓ | ✓ |
| 楽観×現実チャート・KPI | ✓ | ✓ |
| 詳細モード(全パラメータ) | — | ✓ |
| 分析タブ(感度・ストレス・出口最適化) | — | ✓ |
| 運用管理タブ(予実・台帳・申告CSV) | — | ✓ |
| AI市場調査 | — | 月10回 |
| 物件保存 | 3件 | 無制限 |
| 物件比較のIRR・CCR・DSCR列 | 🔒 | ✓ |

プランは `src/plan.js` の `PLANS` で変更できます。

## デプロイ手順(Vercel)

1. このフォルダをGitHubリポジトリにpush
2. https://vercel.com で「New Project」→ リポジトリを選択(設定はデフォルトでOK。ViteとAPIは自動認識)
3. 環境変数を設定: `ANTHROPIC_API_KEY` = Anthropic APIキー
   (これでAI市場調査が `/api/research` 経由で動きます。キーはサーバー側のみで、ブラウザには出ません)
4. Deploy

ローカル確認は `npm install` → `npx vercel dev`(API込み)
または `npm run dev`(画面のみ。AI調査は404になります)。

## 課金フロー(v1: 手動でも回る最小構成)

1. Stripeで「Payment Link」を作成(月額サブスク ¥1,480)
2. そのURLを `src/plan.js` の `PURCHASE_URL` に設定
3. 購入通知が来たら、ライセンスキーを発行して購入者にメール:
   ```bash
   npm run genkey      # 1枚発行
   npm run genkey 10   # 10枚まとめて発行
   ```
4. 購入者はアプリの「Proにアップグレード」→ キー入力で開放

## セキュリティに関する正直な注意

- v1のキー検証は**ブラウザ内のチェックサム照合**です。コードを解析すれば
  突破できるため、本格運用では検証をサーバーレス関数に移すか、
  Stripeのサブスクリプション状態をAPIで照会する方式に置き換えてください。
- `src/plan.js` と `scripts/genkey.mjs` の `SALT` は公開前に必ず独自の値に
  変更してください(両ファイルで同一の値にすること)。
- プラン状態・保存データは端末のlocalStorageに保存されます。端末をまたぐ
  同期にはアカウント機能(認証+DB)の追加が必要です。
